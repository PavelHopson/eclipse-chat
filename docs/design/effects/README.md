# Effect References

This folder contains selected local visual-effect references for Eclipse Chat design exploration.
They are references, not production dependencies.

## Useful Directions

- Deadline / timer effects: task urgency, reminders, action queue and deadline states.
- Interactive galaxy / black-hole effects: landing page, home command center, AI memory visuals.
- Spider clock: experimental time / focus / rhythm motif, use carefully.
- Login and signup forms: auth-screen interaction ideas, adapt to Eclipse tokens before shipping.
- Logout buttons: destructive or session-ending micro-interactions.
- Dark/light toggles: theme switch exploration for VOID / SOLAR modes.

## Rules Before Implementation

- Rebuild effects in production CSS/React instead of copying heavy demo code blindly.
- Keep accessibility and `prefers-reduced-motion`.
- Avoid effects that compete with chat readability.
- One strong ambient effect per screen is enough.
- Validate on mobile before merging into the app.
